# MEASURMENT PARAMETERS

When you select an individual measurement in the left column of the Measurement List configuration window, the configurable properties for that signal appear to the right.

| Parameter | Description |
| ----------- | ----------- |
| Label | The name of this measurement. This will be displayed in the left column of the Measurement List.|
| Measurment | Click this link to select the measurement whose output values will be displayed. |
| Unit | Once the Measurement is selected, use this dropdown to select the units for this parameter. For example, Speed could be in mph or km/h. |
| Decimals | The number of decimal places displayed for this parameter in the Measurement List.|
| Add New Range| Click this button to define a range of values for this measurement that will cause the value to be displayed in a different color. For example, you may wish the value to displayed in red if it's above the safe operating limits for the engine. |
