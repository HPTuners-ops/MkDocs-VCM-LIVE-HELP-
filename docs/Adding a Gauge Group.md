# ADDING A GAUGE GROUP

A gauge group is subwindow that can contain multiple gauges. This allows the group to be moved and resized as a unit.

1. In the menu bar, select **Layout** > **Add Control** > **Add Measurement** > **Add Gauge Group**. An empty gauge group appears.

    ![](gauge_group_empty.png)

2. Add the individual gauges:

    a. Right click inside the empty group. A popup menu that lists gauge types appears.

    b. Select the type of gauge to add: Dial Gauge, Bar Gauge or Value Gauge. A window showing the properties for the new gauge appears.

    c. Configure the parameters for the new gauge. These is identical to configuring the same gauge types as standalone gauges.

    d. Click **Create Window**.
