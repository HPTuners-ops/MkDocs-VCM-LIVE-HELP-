# TIME GRAPH PARAMETERS

When you select an individual signal (measurement) in the left column of Time Graph configuration window, the configurable properties for that signal appear to the right.

| Parameter | Description |
| ----------- | ----------- |
| Label | The name of this signal. This will be displayed in the legend on the left side of the Time Graph subwindow.|
| Measurement | Click this link to select the measurement whose output values will be charted. |
| Unit | Once the Measurement is selected, use this dropdown to select the units for the graphed measurement. For example, speed could be in mph or km/h. |
| Decimals | The number of decimal places displayed for this measurement on the Time Graph subwindow.|
| Min Value | The minimum signal value that will be displayed in the graph. This value will correspond to the bottom edge of the graph. Values below this will be cut off. |
| Max Value | The maximum signal value that will be displayed in the graph. This value will correspond to the top edge of the graph. Values above this will be cut off. |
| Color | The signal's color in the Time Graph subwindow. |
