# THE TABLE EDITOR TOOLBAR

The toolbar in Table Editor has many options for viewing and modifying tables.

![](table_editor_toolbar.png)

## Select View

| Icon | Description |
| ----------- | ----------- |
| ![](3d_chart_view_icon.png) | Selects the 3D graph view. In split view mode, this changes between 2D, 3D and histogram view.|
| ![](2d_chart_view_icon.png) | Selects the 2D graph view. In split view mode, this changes between 2D, 3D and histogram view. |
| ![](histogram_icon.png) | Selects the histogram view. In split view mode, this changes between 2D, 3D and histogram view.
| ![](view_table_icon.png) | Selects the table view. In split view mode, the table is always one of the views in the split.|

## Single View v.s. Split View

| Icon | Description |
| ----------- | ----------- |
| ![](single_view_icon.png) | View only one data representation (table, 2D/3D graph or histogram).|
| ![](vertical_split_icon.png) | Use a vertical split with the table display on the left and the graph or histogram on the right. |
| ![](horizontal_split_icon.png) | Use a horizontal split with the table display on top and the graph or histogram on the bottom. |

## Precision

| Icon | Description |
| ----------- | ----------- |
| ![](increase_precision_icon1.png) | Increase decimal precision (add a decimal place).|
| ![](decrease_precision_icon.png) | Decrease decimal precision (remove a decimal place).|

## Modify

| Icon | Description |
| ----------- | ----------- |
| ![](equals_icon.png) | Replace selected cells with the value in the box.|
| ![](plus_icon.png) | Add the value in the box to the selected cells. |
| ![](times_icon.png) | Multiply the selected cells by the value in the box. |

## Smoothing and Interpolation

| Icon | Description |
| ----------- | ----------- |
| ![](smooth_entire_selection_icon.png) | Smooth the entire selection.|
| ![](smooth_horizontal_icon.png) | Smooth the selection only using a horizontal pass. |
| ![](smooth_vertical_icon.png) | Smooth the selection only using a vertical pass.<br><br>**NOTE:** When using the smoothing functions, the level of smoothing will be determined by your current precision. To increase the effected area of your smoothing, increase your precision
| ![](interpolate_entire_selection_icon.png) | Interpolate the entire selection (flat plane).|
| ![](interpolate_horizontal_icon.png) | Interpolate between horizontal bounds (creates lines). |
| ![](interpolate_vertical_icon.png) | Interpolate between vertical bounds (creates lines). |
