# EDITOR PARAMETERS

When you select an individual table in the left column of Characteristic Editor configuration window, the configurable properties for that signal appear to the right.

There are two or three groups of properties. The top group includes the table title and the format for the cell data.

| Parameter | Description |
| ----------- | ----------- |
| Label | The name of this table. This will be displayed on the tab at the top of the subwindows.|
| Characteristic | Click this link to select the configuration table that you wish to display. |
| Unit | Once the Characteristic is selected, use this dropdown to select the units for the table cell data. For example, Speed could be in mph or km/h. |
| Decimals | The number of decimal places displayed in each of the table's cells.|

The second group configures the horizontal axis.

| Parameter | Description |
| ----------- | ----------- |
| Primary Axis | Displays the name of the horizontal axis (not configurable).|
| Characteristic | Select the units for the horizontal axis. |
| Unit | The number of decimal places displayed on the horizontal axis. |

For two-dimensional tables only, the third group configures the vertical axis.

| Parameter | Description |
| ----------- | ----------- |
| Secondary Axis | Displays the name of the vertical axis (not configurable).|
| Unit| Select the units for the vertical axis.|
| Decimals| The number of decimal places displayed on the vertical axis.|

The bottom group can be used to configure a histogram view for the table.

| Parameter | Description |
| ----------- | ----------- |
| Histogram | Click to select a measurement to use for live histogram data. This is typically a measurement that represents a calculated correction to the base table.|
| Unit | Select the units for the histogram data. |
| Operation | Specifies the default plot for this histogram is maximum value, minimum value, average value or hit count. |
| Log Size | Specifies whether the default view for this histogram is based on the full length of the log or just the "zoomed" time frame represented by the timeline slider bar.|
| Decimals | The number of decimal placed to display for histogram data. |
