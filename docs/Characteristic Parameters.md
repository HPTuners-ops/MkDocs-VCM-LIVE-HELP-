# CHARACTERISTIC PARAMETERS

There are four parameters that can be configured for characteristics in a list.

| Parameter | Description |
| ----------- | ----------- |
| Label | The name of this characteristic. This will be displayed the left column of the list.|
| Characteristic | Click this link to select the characteristic that you wish to include in the list. |
| Unit | Once the Characteristic is selected, use this dropdown to select the units for the table cell data. Although this control is displayed for all characteristics, it only applies to characteristics whose values are numerical.|
| Decimals| The number of decimal places displayed in each of the table's cells. Although this control is displayed for all characteristics, it only applies to characteristics whose values are numerical. |
