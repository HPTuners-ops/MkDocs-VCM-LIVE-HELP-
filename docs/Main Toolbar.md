# MAIN TOOLBAR

VCM Live's main toolbar contains the following items:

![](toolbar.png)

| Icon | Description |
| ----------- | ----------- |
| ![](open_in_new.png) | Open an existing tune file.<br><br> **NOTE:** If you already have a file open, this will also close the currently open file.|
| ![](save_tune_icon.png) | Save the open tune file. |
| ![](reload_icon.png) | Undo the most recent change to the tune. Click again to undo an additional change (up to 50).|
| ![](open_layout_icon.png) | Opens a saved layout of gauges, charts, lists and/or configurable tables. See [Layouts](Layouts.md) for more information.|
| ![](save_layout_icon.png) | Saves the current configuration of displayed gauges, charts, lists and/or tables to a file. See [Layouts](Layouts.md) for more information.|
| ![](open_logfile_icon.png) | Open an existing log file.<br><br> **NOTE:** If you already have a file open, this will also close the currently open file.|
| ![](save_logfile_icon.png) | Save the current recording to a log file.|
| ![](close_logfile_icon.png) | Close the open log file.|
| ![](connect_to_vehicle_icon.png) | Opens a data connection to the ECU.|
| ![](disconnect_from_vehicle_icon.png) | Closes the data connection to the ECU.|
| ![](read_vehicle_icon.png) | If there is an ECU connected to your computer, select this icon to read the ECU's current tune.|
| ![](write_vehicle_icon.png) | If there is an ECU connected to your computer and you have a tune file appropriate for that vehicle open, select this icon to write the tune in the open file to the ECU.|
| ![](dtc_icon.png) | Open the Diagnostics and Information window, which is used to check and clear DTCs. See [Diagnostics & Information](Diagnostic & Information.md) for more information.|
| ![](livetune_icon.png) | If there is an ECU connected to your computer, click this icon to enable live tuning. It will begin rotating to indicate live tuning is active. Click again to disable live tuning.|
| ![](start_scanning_icon.png) | start recording.|
| ![](stop_scanning_icon.png) | Stop recording / Stop playback of log file.|
| ![](play_log_icon.png) | Begin playback of the log file you currently have open.|
| ![](zoom_in_icon.png) | When viewing log file data or actively recording data from an ECU, click this icon to zoom in on the timeline (show a shorter period of time in more detail). See [The Timeline](The Timeline.md) for more information on using the timeline.|
| ![](zoom_out_icon.png) | When viewing log file data or actively recording data from an ECU, click this icon to zoom in on the timeline (show a longer period of time in less detail).|
