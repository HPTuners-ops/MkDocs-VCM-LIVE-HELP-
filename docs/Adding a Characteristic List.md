# ADDING A CHARACTERISTIC LIST

Characteristic lists contain a number of tunable parameters whose values can be adjusted by the user. To add such a list,

1. In the menu bar, select **Layout** > **Add Control** > **Add Characteristic** > **Add List**. The below window will appear.
![](117.jpg)

2. Click ![](add_chart_signal_icon.png) icon and the below window will appear. Search for the characteristic (s) you wish to add.

    ![](select_characteristic.png)

3. Double-click on the characteristic you wish to add.

4. When all the characteristics that you want to include in the list have been added and configured, Click **Create Window**.

## Toolbar

| Parameter | Description |
| ----------- | ----------- |
| ![](add_chart_signal_icon.png) | Add a new characteristic to the list. To complete signal setup, you must configure the characteristic parameters.|
| ![](remove_chart_signal_icon.png) | Delete the characteristic selected in the left column. |
| ![](duplicate_chart_signal_icon.png) | Duplicate the characteristic selected in the column on the left. |
| ![](up_arrow.png) | Move the characteristic selected in the left column upwards in the list.|
| ![](down_arrow.png) | Move the characteristic selected in the left column downwards in the list. |
