# LAYOUTS

Layouts specify the arrangement and content of the subwindows that appear in the main display area of VCM Live. There are two basic elements of a layout:

- **Layout Pages (Tabs)** - an individual layout can contain multiple pages, which are selected using the tabs just below the toolbar. See [Layout Pages (Tabs)](Layout Pages (Tabs).md) for more information on adding, removing and configuring pages.

- **Subwindows** - subwindows representing sensor data or groups of tunable parameters can be added to the layout and then moved, resized, and configured in multiple ways. See [Subwindows](Subwindows.md) for more information.
>**NOTE:** Modifying a layout involves changing these elements. Refer to the relevant sections.

## Saving a Layout

To save the current layout, including subwindow size and position, as well as any customization you have done to the individual subwindows:

1. In the toolbar, select ![](save_layout_icon.png).

2. 	If this is a new layout, you will be prompted to enter a name for the layout. Enter a name for file and click **Save**.

## Loading a Saved Layout

To load a layout you have saved to file:

1. In the toolbar, select ![](open_layout_icon.png).

2. Browse for the saved layout file and click **Open**.

## Loading a Default Layout

There are a number of pre-defined layouts listed in the menu bar. To open one of these layouts:

1. In the menu bar, select **Layout** > **Default layouts** and select either **Initial Setup**, **Basic**, or **Advanced**.
![](new default layout image.png)

2. A list of all the tabs included in the selected layout appears.
![](Advanced Layout image.png)

3. Select the tabs from this layout to load. By default, all the tabs will be loaded. If you wish to remove a tab:

    a. Select the tab to remove in the right column.

    b. Click << to move it to the left column.

4. 	If you wish to reorganize certain tabs:

    a. Select the tab to reorganize of the right column.

    b. 	Click either the ^ or v to reorganize your tabs.

5. Click **Ok**.
