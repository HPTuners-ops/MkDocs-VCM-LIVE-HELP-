This VCM Live Help system is a test run for converting all current documentation into MkDocs with the material theme.

It can be helpful to preview changes on your computer opening a pull request, perform the below steps to preview it.

Ensure you have installed MkDocs on your device
Ensure python is installed as well
follow the folder "cd C:\Users\aavelar\mkdocs"
Point your browser at http://127.0.0.1:8000/
Copy and paste "mkdocs serve --livereload --dev-addr=127.0.0.1:8000" into your command prompt to preview the help system
